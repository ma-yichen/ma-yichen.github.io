## Course Overview

Discrete event simulation methodology emphasizing the statistical basis for simulation modeling and analysis. Overview of computer languages and simulation design applied to various industrial situations.

Topics covered:

- Fit input distributions using data collected from the target system
- Develop and validate simulation models aligned with analysis objectives
- Statistically analyze system performance using simulation outputs
- Compare policies, system designs, or alternatives through simulation and draw statistically sound conclusions
- Derive engineering and business insights from simulation studies to support decision-making under uncertainty
