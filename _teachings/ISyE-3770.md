## Course Overview

Topics covered:

- Understand how to characterize and assess probability in experimental settings
- Use statistical tests and confidence intervals to evaluate uncertainty in statistical decision-making
- Select appropriate statistical techniques based on the type of data available
- Use statistical software and toolboxes to conduct data analysis and interpret results
- Draw sound statistical conclusions from data analysis
